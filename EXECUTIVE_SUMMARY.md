## 💡 Ключевые особенности SmartSell

### Что делает платформу:
1. **Автоматизация продаж на Kaspi** - синхронизация заказов и товаров
2. **Управление каталогом** - товары, категории, варианты, переоценка
3. **Обработка заказов** - от корзины до выполнения
4. **Маркетинг** - SMS/Email кампании с таргетингом
5. **Биллинг** - подписки, кошельки, инвойсы
6. **Склад** - инвентаризация и движение товаров
7. **Аналитика** - отчеты по продажам и заказам

### Технологии:
- **Backend:** Python 3.11+, FastAPI, SQLAlchemy, PostgreSQL, Redis
- **Frontend:** React 18, TypeScript, Vite, Material-UI
- **Infrastructure:** Docker, Docker Compose, Nginx, Let's Encrypt
- **Testing:** pytest (140+ tests, 90% coverage)
- **CI/CD:** GitHub Actions (lint, test, security)

## 🔒 Безопасность

### ✅ Реализовано:
- JWT аутентификация (HS256/RS256 с rotation)
- RBAC v2 (Platform/Store roles с гибкими правами)
- Пароли: Argon2 (primary), проверка на слабые пароли
- HTTP Security: CORS, security headers, CSRF, rate limiting
- Audit logging: полное журналирование действий
- Tenant isolation: изоляция данных по компаниям
- Шифрование: pgcrypto для sensitive data

### ⚠️ Рекомендации:
- Настроить Sentry для error tracking
- Усилить rate limiting на критичных эндпоинтах
- Добавить Prometheus + Grafana для мониторинга

**Вывод:** Безопасность на высоком уровне, готова к продакшену

## 🏗️ Архитектура

```
┌─────────────┐      ┌─────────────┐      ┌──────────────┐
│   Frontend  │─────▶│  Nginx      │─────▶│   FastAPI    │
│  (React)    │      │  (Reverse   │      │   Backend    │
│             │      │   Proxy)    │      │              │
└─────────────┘      └─────────────┘      └───────┬──────┘
                                                   │
                     ┌─────────────────────────────┼────┐
                     │                             │    │
                     ▼                             ▼    ▼
              ┌──────────┐                  ┌──────────────┐
              │PostgreSQL│                  │    Redis     │
              │          │                  │(Cache/Queue) │
              └──────────┘                  └──────────────┘
                     │
                     │  Celery Workers
                     ▼
              ┌──────────────┐
              │  Background  │
              │    Tasks     │
              └──────────────┘
                     │
                     │  External APIs
                     ▼
        ┌──────────────────────────┐
        │  Kaspi, TipTop, Mobizon  │
        │  Cloudinary, SMTP, etc   │
        └──────────────────────────┘
```

### Ключевые паттерны:
- **API Versioning:** Clean `/api/v1/` структура
- **Service Layer:** Бизнес-логика отделена от API
- **Repository Pattern:** Абстракция доступа к данным
- **Dependency Injection:** FastAPI dependencies
- **Background Tasks:** Celery для async операций
- **Event Sourcing:** Integration events для аудита
